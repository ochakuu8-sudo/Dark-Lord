# ユニット設計書

このファイルを編集してClaude Codeに渡すと、`entities.ts` / `config.ts` に反映してもらえます。

---

## 凡例

- **素材**: 🦴骨(0) / 🥩肉(1) / 🔮霊(2) / ❓ワイルド(9)
- **レア度**: common / rare / legend
- **attackType**: melee / ranged
- **size**: ピクセル半径（レンダリング・ヒットボックス）
- **accuracy**: 命中率 0〜1（現在全ユニット1.0）
- **passive**: 未実装はすべて `-`

## パッシブ能力リスト（実装済み）

現在全ユニットのパッシブは取り外し中。以下は実装済みの型定義のみ残存。

| passive ID | 説明 |
|-----------|------|
| REFLECT | ダメージの一部を攻撃者に反射 |
| AURA_REGEN | 周囲の味方HPを毎秒回復 |
| PIERCING | 攻撃が敵を貫通 |
| PIECE_RETURN | 死亡時に広範囲のピースを回収 |
| PROXIMITY_EXPLOSION | 敵が近づくと自爆 |
| ATK_BUFF | 周囲の味方の攻撃力を上昇 |
| HEAL_SHOT | 最も傷ついた味方を回復 |
| SUMMON | 一定時間ごとにゾンビ召喚 |
| CORPSE_EXPLOSION | 近くの味方の死で爆発 |
| INSTANT_AOE | 敵の位置に直接範囲魔法 |
