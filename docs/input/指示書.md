# Dark-Lord 実装指示書（現行ユニット反映版）

## 前提
- この指示書は、現在のユニット一覧に書かれている効果をそのまま実装へ落とすためのもの。
- **実装対象は、実際にユニット効果で使われている語だけ**に絞る。
- 以前の案にあったが、**今のユニット効果に出てこない定型句はまだ実装しない**。
- 定型句の意味づけも、まずは現在のユニット効果を説明できる最小限にする。

## 1. 味方ユニット効果を現行内容に合わせて反映する

### コモン

#### ゴブリン
- 骨: 攻撃力が高い
- 肉: 体力が高い
- 霊: 通常攻撃が爆発

#### スケルトン
- 骨: 「召喚時」自身を追加で2体「召喚」する
- 肉: 「死亡時」骨スケルトンを「召喚」する
- 霊: 「常時」「ステルス」（体力は1）

#### アーチャー
- 骨: CDが遅い代わりに一撃が強い。
- 肉: 「3連射」前方120度のどこかにランダムに攻撃を飛ばす。
- 霊: 「攻撃時」弾を「貫通」させる

#### オーク
- 骨: 「被弾時」攻撃者へ「20%反射」する
- 肉: 「体力低下時」自身に「攻撃力上昇」を100%付与
- 霊: 「攻撃時」前方攻撃を「範囲化」する + 「常時」自身に「再生」を「付与」する

#### リッチ
- 骨: 「攻撃時」弾を「跳弾」させる
- 肉: 「X秒毎に」周囲の味方を「回復」する
- 霊: 「命中時」着弾地点を「爆発」させる

#### ケルベロス
- 骨: 「攻撃時」攻撃を「連射」する
- 肉: 「命中時」与ダメージの一部を自身へ「回復」する
- 霊: 「命中時」対象に「毒」を付与する

#### ウィスプ
- 骨: 「死亡時」周囲に「爆発」する
- 肉: 「死亡時」周囲の味方を「回復」する
- 霊: 「死亡時」周囲に「スタン」を付与する

### レア

#### ネクロマンサー
- 骨: 「1秒毎に」骨スケルトンを「召喚」する
- 肉: 「敵が死亡時」味方を最大体力の20%「範囲回復」する
- 霊: 「味方が死亡時」周囲へ「爆発攻撃」する

#### ミノタウロス
- 骨: 「常時」走り続けて、ぶつかった相手に100%攻撃力ダメージ
- 肉: 「戦闘開始時」、水平に敵の後ろの壁に向かって突進する。遠距離耐性 50%
- 霊: 「攻撃時」一定距離突進する。突進でぶつかった相手全員にダメージとノックバック

#### グール
- 骨: 「一定時間毎に」骨グールを召喚する。HPが低い
- 肉: 「50%回避」「回避時」「回復」
- 霊: 「被弾時」自身の周りに「爆発」し、爆発が当たった相手に「毒」を付与

## 3. 各定型句の最小定義
未確定のものは無理に広げず、今のユニット効果を成立させる範囲だけ定義する。

### トリガー定義
- 「召喚時」: 召喚が起こった時
- 「死亡時」: そのユニットが死亡した時に発動
- 「常時」: 常に有効
- 「攻撃時」: 通常攻撃を行う時に発動
- 「被弾時」: 通常攻撃を受けた時に発動
- 「命中時」: 通常攻撃が相手に当たった時に発動
- 「戦闘開始時」: 戦闘開始時に1回発動
- 「突進終了時」: 突進動作の終了時に発動
- 「X秒毎に」: 一定秒数ごとに繰り返し発動
- 「敵が死亡時」: 敵ユニットが死亡した時に発動
- 「味方が死亡時」: 味方ユニットが死亡した時に発動
- 「一定時間毎に」: 秒数未定の周期発動
- 「被攻撃時」: 攻撃対象にされた時、または攻撃を受けた時の判定

### 効果 / 処理定義
- 「召喚」: 戦闘開始以外で新しいユニットを場に出る
- 「付与」: 状態や性質を対象に与える
- 「回復」: HPを回復する
- 「範囲回復」: 範囲内の味方をまとめて回復する
- 「爆発」: 周囲または指定地点に範囲ダメージを発生させる
- 「爆発攻撃」: 爆発による攻撃扱いのダメージ
- 「突進」: 高速移動して指定方向または指定対象へ進む
- 「ノックバック」: 対象を後ろへ押し戻す
- 「連射」: 通常攻撃がCDごとに複数回攻撃する
- 「貫通」: 攻撃が複数体に当たる
- 「反射」: 受けたダメージの一部を攻撃者に返す
- 「跳弾」: 攻撃ヒット後、その攻撃が別の敵へ飛んでいく

### 状態 / 性質定義
- 「ステルス」: 他のユニットが全滅するまで狙われない
- 「再生」: 時間経過でHPを回復する
- 「毒」: 継続ダメージを与える。スタック可。一定時間受けなければ0になる
- 「〇〇耐性」: 特定のタグの受けるダメージを軽減する
- 「回避」: 攻撃を一定確率で無効化する
- 「スタン」: 攻撃も移動もできない

### 対象 / 範囲定義
- 「自身」: 発動したユニット本人
- 「対象」: 現在攻撃している相手
- 「味方」: 味方ユニット
- 「周囲」: ユニットの周辺円形範囲
- 「前方」: ユニット正面方向の範囲
- 「着弾地点」: 攻撃が当たった位置
- 「攻撃者」: そのダメージを与えた相手
- 「ぶつかった相手」: 突進や移動で接触した相手
- 「相手全員」: 接触した複数対象全体
- 「敵の後ろの壁」: 敵列の奥側にある壁位置

## 4. バフは、現在の実装で使っている値だけを強化対象にする
新しい概念は増やさず、今ある値にだけ乗るバフを用意する。

### 今回追加してよいバフ
- 「攻撃力上昇」
- 「最大体力上昇」
- 「射程上昇」
- 「回復量上昇」
- 「再生量上昇」
- 「毒ダメージ上昇」
- 「遠距離耐性上昇」
- 「回避率上昇」
- 「召喚数上昇」
- 「召喚間隔短縮」
- 「爆発ダメージ上昇」
- 「爆発範囲上昇」
- 「突進距離上昇」
- 「ノックバック距離上昇」
- 「連射回数上昇」

### バフ定義
- 「攻撃力上昇」: 通常攻撃、接触ダメージ、攻撃力参照ダメージを強化する
- 「最大体力上昇」: 最大HPを増やす
- 「射程上昇」: 攻撃可能距離を伸ばす
- 「回復量上昇」: 回復・範囲回復・吸収の回復量を増やす
- 「再生量上昇」: 再生による回復量を増やす
- 「毒ダメージ上昇」: 毒の継続ダメージを増やす
- 「遠距離耐性上昇」: 遠距離ダメージ軽減率を上げる
- 「回避率上昇」: 回避発生率を上げる
- 「召喚数上昇」: 1回の召喚で出る数を増やす
- 「召喚間隔短縮」: 周期召喚の間隔を短くする
- 「爆発ダメージ上昇」: 爆発のダメージを増やす
- 「爆発範囲上昇」: 爆発の半径を広げる
- 「突進距離上昇」: 突進の移動距離を伸ばす
- 「ノックバック距離上昇」: 押し戻す距離を伸ばす
- 「連射回数上昇」: 1回の連射で出る攻撃回数を増やす
